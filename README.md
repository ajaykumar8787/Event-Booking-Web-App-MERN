# 🚀 Event-Booking-Web-App (MERN)

A high-performance **Full-Stack MERN** application designed for efficient event management and a seamless booking experience. Users can browse events, book tickets, and administrators can manage event listings with ease.

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind CSS, react
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **Authentication:** JWT (JSON Web Tokens), Bcryptjs
- **Real-time:** Socket.io (for live dashboard updates)

## 🔑 Key Features
* **Authentication:** Secure registration and login system implemented with JWT.
* **Role-based Access:** Distinct permissions for normal users and administrators.
* **Event Management:** Admins can create, update, and delete events.
* **Real-time Booking:** Live booking status updates using Socket.io.
* **Responsive UI:** Fully responsive design optimized for mobile and desktop devices.

📊 Dashboard Overview

* The dashboard provides users and admins with live insights:
* Total Events: Number of active events currently live.
* Booking Analytics: Real-time tracking of confirmed vs. pending bookings.
* Live Updates: Instant notifications and data synchronization via Socket.io.
